# Changelog

All notable changes to this project are documented in this file.

## 0.1.0 - 2026-03-19

### Added

- Release-ready npm metadata (`keywords`, `engines`, `sideEffects`, publish assets).
- Bilingual documentation (`README.md` in English and `README.zh-CN.md` in Chinese).
- API reference document at `docs/API.md`.
- Public API TSDoc coverage for exported types and functions.
- Type contract test coverage for public API usage and invalid combinations.
- `release:check` script to run quality and package dry-run checks.

### Changed

- Documentation clarified collaborative initialization rules and provider expectations.
- Version bumped from `0.0.0` to `0.1.0`.
